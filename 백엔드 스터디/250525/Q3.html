🎴 문제 3: 카드 컴포넌트

목표: 아래 div를 카드처럼 보이게 스타일링하고, 마우스 호버 시 살짝 위로 올라가는 효과를 추가하세요. 현대적인 UI 카드를 만들어보세요!

멋진 카드 제목

이것은 카드 컴포넌트입니다. 마우스를 올려보세요! 부드럽게 위로 올라가면서 그림자가 더 진해질 거예요.

카드는 현대 웹 디자인에서 자주 사용되는 UI 요소입니다.

💡 힌트

기본 카드 스타일:

  • background-color: white; - 흰색 배경
  • padding: 20px; - 내부 여백
  • border-radius: 8px; - 둥근 모서리
  • box-shadow: 0 2px 8px rgba(0,0,0,0.1); - 그림자
  • max-width: 400px; - 최대 너비 제한
  • transition: all 0.3s ease; - 부드러운 애니메이션

호버 효과:

  • transform: translateY(-5px); - 위로 5px 이동
  • box-shadow: 0 8px 25px rgba(0,0,0,0.15); - 더 큰 그림자

🚀 도전 과제

기본 문제를 완료했다면:

  • 카드에 border를 추가해보세요
  • hover 시 배경색이 살짝 바뀌게 해보세요
  • 카드 내부의 제목 색상도 hover 시 변경해보세요
  • cursor: pointer;를 추가해서 클릭 가능해 보이게 만들어보세요
