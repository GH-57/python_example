🧭 문제 2: 네비게이션 메뉴

목표: 아래 리스트를 가로로 배치하고, 각 메뉴 항목에 padding과 hover 효과를 추가하세요. 깔끔한 네비게이션 메뉴를 만들어보세요!
  • 홈
  • 소개
  • 서비스
  • 연락처

💡 힌트

nav-menu (ul 요소):

  • display: flex; - 가로 배치 (또는 display: inline-block을 li에 사용)
  • list-style: none; - 리스트 점 제거
  • padding: 0; margin: 0; - 기본 여백 제거

nav-menu li:

  • margin-right: 20px; - 메뉴 간격

nav-menu a:

  • text-decoration: none; - 밑줄 제거
  • padding: 10px 15px; - 클릭 영역 확대
  • color: #333; - 텍스트 색상
  • border-radius: 4px; - 둥근 모서리

hover 효과:

  • background-color: #007acc;
  • color: white;
